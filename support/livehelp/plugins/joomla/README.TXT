Joomla Module Instructions
----

1. Login to the Joomla Administration
2. Browse to Extensions > Install / Uninstall
3. Type the full path to the /livehelp/plugins/joomla/mod_livehelp directory within the "Install from Directory" option and press the Install button
4. Enable the Live Help Module within the Extensions > Module Manager > Site
5. Customise the Live Help Module Position / Order etc.
6. Finished
