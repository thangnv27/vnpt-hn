WordPress Plug-in Instructions
----

1. Copy the /livehelp/plugins/wordpress/livehelp/ directory to your wp-content/plugins/ directory
2. The plugin directory should now be located at wp-content/plugins/livehelp/
3. Activate the Live Help plugin at WordPress Administration > Plugins > Installed
4. Add the Live Help widget to your template at WordPress Administration > Appearance > Widgets
5. Finished
